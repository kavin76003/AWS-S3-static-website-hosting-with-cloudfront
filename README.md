# AWS S3 Static Website Hosting Project

## Overview
This project shows how to host a simple static website using **Amazon S3** and improve performance using **CloudFront**.

The website contains basic HTML and CSS files and is publicly accessible through a URL.

---

##  Architecture
User → CloudFront → S3 Bucket → Website Files

---

##  Services Used
- Amazon S3
- AWS CloudFront

---

## Steps

1. Create an S3 bucket
2. Upload `index.html` and `style.css`
3. Enable static website hosting
4. Add bucket policy for public access
5. Create CloudFront distribution
6. Access website using CloudFront URL

---

##  Output
- Website hosted on S3
- Faster access using CloudFront
